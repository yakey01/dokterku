Masalah:
Browser modern tidak mengizinkan akses GPS/Geolocation melalui koneksi HTTP yang tidak aman. Anda mengakses dari http://10.103.117.197

üîß Solusi yang Tersedia:

Option 1: Gunakan Localhost (Recommended untuk Testing)

Akses aplikasi melalui localhost yang diizinkan untuk development:

http://localhost:8000/dokter/mobile-app
atau
http://127.0.0.1:8000/dokter/mobile-app
Buka di Localhost

Option 2: Setup HTTPS dengan Laravel Herd

Jika menggunakan Laravel Herd, Anda bisa enable HTTPS:

# Di terminal, jalankan:
herd secure dokterku

# Kemudian akses:
https://dokterku.test

Option 3: Gunakan Ngrok untuk HTTPS Tunnel

Buat HTTPS tunnel untuk development:

# Install ngrok jika belum ada
brew install ngrok

# Jalankan tunnel
ngrok http 8000

# Akan mendapat URL seperti:
https://abc123.ngrok.io

Option 4: Testing dengan Default Location

Untuk testing sementara, aplikasi akan menggunakan lokasi default Klinik Dokterku jika GPS tidak tersedia.

  • Latitude: -7.8481
  • Longitude: 112.0178
  • Location: Kediri, Jawa Timur

üîí Kenapa GPS Memerlukan HTTPS?

Browser membatasi akses ke API sensitif seperti:

  • Geolocation/GPS
  • Camera
  • Microphone
  • Device Motion

Ini adalah fitur keamanan untuk melindungi privasi pengguna dari website yang tidak aman.
